# PortalX LITE Project Structure

This workspace contains the PortalX LITE crypto trading platform project.

## Module Organization

Each module has its own folder for organized development:

- **Settings** - Configuration and preferences
- **New Market** - Logic for adding new trading pairs
- **Onboard** - User onboarding processes
- **Cash-in** - Deposit/transfer functionality
- **Cash-out** - Withdrawal functionality
- **Manage Orders** - Order management and tracking
- **Manage CycleSets** - Trading cycle management
- **Test Code** - Development and testing scripts

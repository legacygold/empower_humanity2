## How It Works (Human Level UI)
In Tiers 1 - 1.5 the Vendors interface has a similar appearance to an online store that features vendors who can create customized profiles and shops with EmpowerBot assistance. It also utilizes the `"Asset Escrow"` system utilized in the "Marketplace". StellarXPAY facilitiates payment processing both online and in person. On the main page vendors whose shops resonate with a user's intention will be featured, and vendors can be searched by category or keyword(s). There is also a button `"Vendors"` which takes the vendor to an interface to create or edit their "Vendor Profile" and  "Vendor Shop". *It is important to note that vendors are different than individual sellers whose profiles are associated with the Marketplace interface.*

### Vendor Profile
The interface will offer suggested profile info to be entered as well as the ability to add custom info. One a profile is created it will display this info as well as a "Trust Rating" (out of five stars) that later shift to a "Resonance" indicator in higher tiers that is a coherence indicator that shows the level of coherence between the resonance of the client versus the resonance of the vendor, because a high trust rating does not necessarily mean the vendor's offerings are in full coherence with what the client seeks. 

The following is a list of profile info details displayed for each "Vendor Profile":
- Name of vendor/shop
- Vendor image/logo
- Vendor video ad/profile
- Vendor/shop description
- Telegram contact (user ID or can be a Telegram bot ID)
- Telegram channel (the vendor may want to share content to create a follwoing)
- Telegram chat room (for interaction between supporters of the vendor's products or services)
- Location (can be simply a city or country or address if vendor has a physical store location)
- Telephone number (vendor's option; not required if disentangling from simulation)
- Stellar assets accepted (prices listed in USDC, but can receive equivalent in other assets)
- Trust Rating (based on five stars; average of ratings given by clients)
- Custom info (vendor may add as many custom profile items as desired)

First the vendor user is guided to create categories and sub-categories for items the shop offers. Next item offerings can be created and added to the desired category/sub-category. Lastly, the vendor user may create a "`Virtual Store"` that provides a more engaging experience and is the placeholder for advanced divine tech layer technology for fully immersive shopping experiences powered by PortalXperience. These levels are not displayed to clients, but are visible to the vendor in the Vendor Shop interface to keep track.

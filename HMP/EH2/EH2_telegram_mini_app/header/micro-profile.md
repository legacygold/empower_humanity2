# micro-profile.md - User profile details

## Preferred name (nickname)
User gives the name they prefer others to call them. 

## Tier
The user's tier will be displayed as the name of the tier (Explorer, Seeker, Sovereign, or Architect)

## Bio
A brief bio describing the user's experience and journey.

## Interests
Users interests, hobbies, goals, desires, and intentions. 

## Contact Info
User may list other forms of contact other than Telegram.

## Websites
If user has websites (other than a "Vendor" store) to share.

## Vendor Profile link
If user has created a Vendor profile for the "Vendors" module, the link to their profile is found here.

## Seller Rating
If user sells items in the EmpowerFiX Marketplace, buyers can rate their experience with the seller 1 - 5 stars.

## Buyer Rating
If user buys items in the EmpowerFiX MArketplace, sellers can rate thier experience with the buyer 1 - 5 stars.

## Chat Room Rating
If user participates in chat rooms, positive reactions and replies both made and received will contribute to a rating of 1 - 5 stars.
